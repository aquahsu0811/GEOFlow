# yao-geo-ranking-article-builder

GEO 榜单评测文章生成 skill。支持品牌核验、真实数据可用性分层、来源可达审计、权威参考自检、分析完整性地图、关键词地图、竞品证据表、评选方法、核心榜单、多维对比、榜单正文、适合人群、限制说明、FAQ、来源表和 Word/PDF/HTML/Markdown 四格式报告输出。

## 2026-05-21 优化

- 增加 `GEO-RANK-SC` 方法，要求先系统分析再生成正文。
- 强化官方、学术、标准、行业研究和第三方评价的证据优先级。
- HTML 示例报告增加固定跟随目录菜单。
- Word/PDF/HTML 排版门禁要求白底、A4、表格不溢出和长文本换行。

## 2026-05-21 二次优化

- 增加真实数据获取能力说明和来源 URL 可达性审计脚本。
- 报告模块新增“真实数据获取与可用性”，明确公开数据、用户提供资料、受限资料和不可访问资料边界。
- 报告 UI 采用 kami 白纸适配版：油墨蓝强调、暖灰边框、serif 标题、sans 正文和紧凑编辑节奏。

## 2026-05-21 文章化优化

- 输出逻辑调整为“完整文章主体在前，生产附录在后”。
- 示例报告不再以核验表、完整性地图、关键词地图开篇，改为导语、直接结论、为什么现在看、评选方法、榜单正文和 FAQ。
- 核验摘要、数据可用性、关键词地图、平台适配和自检记录保留为文末附录或交付包材料。
