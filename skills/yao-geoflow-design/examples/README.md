# yao-geoflow-design examples

This skill edits or clones GEOFlow Laravel Blade themes through a preview-first workflow, so realistic examples usually require a live GEOFlow Laravel workspace.

Use this directory for sanitized theme-edit examples. A publishable example should include:

- target theme identifier and edit mode: `edit_theme`, `clone`, or `hybrid`
- preflight summary from theme discovery
- changed theme files or a preview package summary
- before and after screenshots when they do not expose private project data
- finalization decision: keep preview, publish as new theme, or replace base theme after review

Do not commit private Blade themes, unpublished customer designs, credentials, customized admin paths, or live production activation records.
