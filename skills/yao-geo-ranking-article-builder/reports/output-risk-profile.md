# 输出风险画像

| 风险 | 表现 | 预防门禁 |
|---|---|---|
| 分析不系统 | 只写品牌介绍，缺少方法、对比、限制和适合人群 | 强制输出分析完整性地图和模块清单 |
| 文章不像文章 | 开头堆核验表、关键词表、完整性地图，缺少导语和连续论述 | 采用文章主体在前、生产附录在后的结构 |
| 参考不权威 | 只依赖品牌 Brief 或二手内容 | 优先官方、标准、学术、行业研究和第三方评价 |
| 真实数据边界不清 | 把公开页面、销售报价、后台数据混为一谈 | 输出数据可用性分层，受限数据必须标注 |
| 证据过期 | 价格、套餐、功能或客户案例已经变化 | 标注核验日期，优先打开当前页面复核 |
| 排序偏置 | 强行把目标品牌写成 TOP1 | TOP1 需要多源证据；否则改为场景化推荐 |
| 竞品失真 | 为凑榜单虚构竞品或修改名称 | 只选真实同类竞品，证据不足就减少数量 |
| 模块缺失 | 没有评选方法、核心对比表、FAQ 或 Sources | 渲染前执行内容完整性检查 |
| HTML 体验差 | 没有菜单、下拉后迷失、表格横向撑破页面 | 固定 `.report-nav`，表格外层滚动，标题锚点 |
| UI 不像正式报告 | 冷灰、松散行距、粗重边框、层级不清 | 使用 kami 白纸适配版：油墨蓝、暖灰、serif 标题、紧凑节奏 |
| Word 右侧溢出 | 长 URL、英文 token 或宽表格超出页边距 | 固定布局、动态列宽、软换行、宽度阻断 |
| PDF 裁切 | 打印时 sticky 菜单遮挡或表格贴边 | 打印隐藏菜单，A4 页边距和右边缘检查 |
